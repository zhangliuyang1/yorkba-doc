<h2 id="1">1.登陆</h2>

请求方法：域名/yorkba/app/user/login

支持格式：JSON

http请求方式：POST

应用请求参数：

|参数名称|参数含义|
|---|---|
|userName|【必填】账号|
|passWord|【必填】密码|

返回结果示例：
```
{
  "code":0,
  "message":"",
  "data":{
    "userId":10,
    "accessToken":""
  }
}
```

返回结果说明：

|字段名称|字段含义|
|---|---|
|code|错误标识码，0表示正确|
|message|提示信息|
|data - userId|用户id|
|data - accessToken|用户授权令牌|
